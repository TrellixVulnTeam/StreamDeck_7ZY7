## Installation

Windows
  1. Download [``StreamDeck.zip``](https://github.com/philliphqs/)
  2. Extract ``StreamDeck.zip``
  3. Press Win + R and type ``shell:startup``
  4. Put the ``StreamDeck-Server.exe`` in the folder
  5. Restart your Computer
